Learn more

What is Cloud Z Composer?

Cloud Z Composer는 멀티 클라우드 인프라를 자동으로 Provisioning 하고 설정, 어플리케이션 설치 등을 Single-Dashboard에서 가능하게 해주는 오픈소스로 구성된 Dedicated 플랫폼입니다. Ansible, Ansible AWX, Terraform, Packer, Spinnaker 다양한 Cloud Platform 과의 Integration을 통해 멀티 클라우드 프로비저닝과 관리를 할 수 있습니다.

개발주기의 단순화

프로덕션 수준의 코드를 빠르게 프로토타이핑 할 수 있으며, 비즈니스 로직에만 집중하여 개발 속도가 향상 및 전체 개발 주기를 단순화

비용절감

함수가 실행되는 만큼만 지불하고, 테스트를 통해 자원을 최적화하며 요청이 없을 경우 코드가 실행되지 않으므로, 비용이 최소화
